# Personal Portfolio Website

A modern, responsive, and dark-themed portfolio website built using HTML, CSS, and JavaScript. The website showcases my education, experience, technical skills, projects, and contact information in a clean and professional layout.

## Features

* Modern dark-themed UI
* Responsive design for desktop and mobile devices
* Animated hero section with typing effect
* Education and experience sections
* Skills displayed as interactive tags
* Project showcase with GitHub links
* Resume download functionality
* Contact and social media links
* Smooth scrolling and scroll animations

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts
* AOS (Animate On Scroll)

## Sections

### Hero

Introduction, role highlights, social links, and resume download.

### About Me

Overview of my academic background, interests, and career goals.

### Education

Academic qualifications and institutions attended.

### Experience

Internships and professional experiences.

### Skills

Technical skills related to programming, machine learning, data science, and web development.

### Projects

Highlighted projects with descriptions and GitHub repository links.

### Contact

Email, LinkedIn, and GitHub profile links.

## Deployment

This website is deployed using GitHub Pages.

To run locally:

```bash
git clone https://github.com/GokulKrishnaR/portfolio.git
cd portfolio
```

Open `index.html` in your browser.
